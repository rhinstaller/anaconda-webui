Storage configuration
---------------------

.. figure:: images/step-storage-configuration.png
   :width: 700px
   :align: center
   :alt: Storage configuration

This screen appears after you choose a storage scenario that uses automatic or guided partitioning (not mount point assignment). It allows you to configure automatic partitioning options, disk encryption, and related storage settings.

You can enable or disable encryption for the installation. If you choose to encrypt your storage, you will be prompted for a passphrase. The passphrase may be evaluated for strength; choose a strong passphrase and keep it safe—if you lose it, encrypted partitions and their data cannot be recovered.

.. note::
   No permanent changes are made to your disks until you start the installation (on the Review and install screen). The configuration you set here is only applied when the installation actually runs.
