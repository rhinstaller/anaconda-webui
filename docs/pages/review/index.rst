Review and install
------------------

.. figure:: images/step-review.png
   :width: 700px
   :align: center
   :alt: Review and install

The Review and install screen lets you review your installation settings and start the installation process.

Review the summary of your choices (language, time, storage, users, etc.). When everything is configured as you want, use the button to begin installation. **Nothing will be written to your hard drive before you press it.** You can go back and change settings, or quit the installer at any point before starting; quitting will discard your changes and reboot the system.

Once you start the installation, you cannot go back to change the settings you configured earlier; if you need to change something after the installation has started, you must wait for it to finish, reboot, and adjust settings on the installed system.
