Installation progress
--------------------

.. figure:: images/step-progress.png
   :width: 700px
   :align: center
   :alt: Installation progress

The Installation progress screen is displayed after you finish configuring all required items and start the installation. After this point, the installation process actually starts and changes are being made to your selected disks. It is not possible to go back and change any settings configured earlier; if you need to do so, you must wait for the installation to finish, reboot your system, and change settings on the installed system.

The screen shows a progress indicator and messages informing you of the current progress of the installation. When the installation finishes, you can press the button to reboot your computer and log in to your newly installed system.

.. note::
   Before you finish the installation and reboot, either remove the media (CD, DVD or USB drive) you used to start the installation, or ensure that your system boots from the hard drive before removable media. Otherwise, the computer may start the installer again instead of the installed system.
